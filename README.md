# Tic-Tac-Toe-Game
This is a Tic-Tac-Toe Game built using C programming language.  

Two players (X and O) take turns marking their symbol on a 3x3 grid.  
The player who succeeds in placing three marks in a horizontal, vertical or diagonal row wins the game.
